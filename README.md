# Análisis de la Presión Arterial Sistólica (PAS)

## Objetivo
Evaluar si el tratamiento A reduce la presión arterial sistólica (PAS) en comparación con el tratamiento B, considerando variables como edad, sexo, IMC, colesterol, tabaquismo y diabetes.

## Estructura del proyecto
- Datos/: datos originales y procesados.
- Guiones/: scripts de limpieza, exploración y análisis.
- Salida/Figuras/: gráficos generados.
- Salida/Tablas/: tablas de resultados.

## Instrucciones
1. Colocar el archivo CSV en Datos/raw/.
2. Ejecutar los scripts en orden:
   - 01_limpieza.R
   - 02_exploracion.R
   - 03_modelo_regresion.R
3. Revisar los resultados en la carpeta Salida.
